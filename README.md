# 독서 학원 회원 관리 시스템

독서 학원에서 회원 등록, 출결 관리, 단계 관리, 독서 기록, 결제 상태, 상담 기록을 통합 관리하는 웹 애플리케이션입니다.

## 주요 기능

### 회원 관리
- 회원 등록 / 수정 / 삭제
- 이름, 학년, 전화번호, 레벨 정보 저장

### 출결 관리
- 날짜별 출결 기록 (출석 / 결석 / 지각)
- 출석률 자동 계산 (출석=1, 지각=0.5, 결석=0)

### 독서 관리
- 읽은 책 기록 (제목, 날짜, 이해도, 메모)
- 이해도 분류 (어려움 / 보통 / 잘 이해함)

### 결제 관리
- 결제 상태 관리 (완료 / 미납)
- 금액 및 납부 날짜 기록
- 30일 경과 시 자동 미납 처리

### 상담 기록
- 날짜와 내용 저장
- 대시보드에서 최근 상담 확인

### 필터 및 검색
- 이름 / 전화번호 검색
- 레벨별 필터 (1-9단계)
- 학년별 필터 (초1-고3)
- 특수 필터: 미납 회원, 독서 기록 없음, 이해도 낮음

### 대시보드
- 총 회원 수
- 레벨별 인원 분포
- 평균 출석률
- 미납 인원 수
- 최근 상담 기록
- 학년별 인원 차트

### 자동화 기능
- 학년 자동 상승 (365일 경과 시)
- 결제 미납 자동 처리 (30일 경과 시)

## 기술 스택

- **언어**: 순수 JavaScript (Vanilla JS)
- **데이터 저장**: localStorage
- **UI**: 모바일 + 데스크톱 반응형 디자인
- **프레임워크**: 미사용

## 사용법

1. **회원 등록**: 상단 폼에서 이름, 학년, 전화번호, 레벨을 입력하고 "회원 등록" 클릭
2. **회원 선택**: 좌측 회원 리스트에서 회원 카드 클릭
3. **출결 관리**: 상세 화면에서 날짜와 상태 선택 후 "추가"
4. **독서 기록**: 책 제목, 날짜, 이해도, 메모 입력 후 "추가"
5. **상담 기록**: 날짜와 내용 입력 후 "추가"
6. **결제 관리**: 상태, 금액, 날짜 입력 후 "저장"
7. **필터링**: 좌측 필터로 원하는 조건 선택
8. **대시보드**: 상단 "대시보드" 버튼 클릭
